# Teste Técnico - QA Tester (4blue)

Este repositório contém a análise de testes realizada no microssistema disponibilizado para o processo seletivo da 4blue.

## Sistema testado

https://qa-play-sim.lovable.app/

## Objetivo

Identificar:

- Bugs funcionais
- Problemas de experiência do usuário
- Problemas básicos de segurança

## Estrutura

Os bugs encontrados serão documentados neste repositório.
